# 博客nodejs版 for thinkjs 2.x

> 3.0

## install

```bash
git clone https://github.com/xuexb/blog.git
cd blog
npm install
npm run start
```

## sql cache list

* `new_article_data` 最新文章列表top6
* `list_data` 列表数据
* `search_hit_data` 搜索排行top6
* `rand_tags_list_data` 随机标签列表top20
* `index_article_data` 主页文章列表
