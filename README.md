# 博客nodejs版

使用`thinkjs`创建的博客


## 分支说明

* `master` 主干，为正式线上使用，定期的修复完bug后合并
* `dev` 开发分支，时刻更新bug
* `fix-bug*` 修复bug分支，本地修复，不会同步到github上

## 命令

### jshint

```shell
$ grunt jshint
```

### 编译压缩

> 注：代码必须通过`jshint`检查才可以编译

```shell
$ grunt build
```
